I. Installation

Gen-O-Mix est un logiciel en ligne basé sur Node.JS. Une partie du code est exécutée sur un serveru (service web, générateur de sujets aléatoires...) et doit par conséquent être installé si vous désirez l'utiliser sur votre propre servuer.
Pour ce faire, vous devez simplement télécharger le code source depuis le dépôt de code GIT.

Une fois que vous avez le code installé sur votre machine, il ne vous reste qu'à lancer la commande

node bin/www

Vous pouvez ensuite vous connecter à votre serveur au travers de votre navigateur web en tappant l'URL http://votre_address_IP:3000/

III. Effectuer une simulation


Interface Gen-O-Mix

III.1. Description de l'interface

  1. Réinitialiser la simulation : Ce bouton permet de réinitialiser la simulation en effaçant les résultats des précédentes générations,
  2. Simuler la prochaine génération : Selon l'état actuel de la simulation:
    • Simulation non initialisée : Génère la première génération composée d'un nombre égal d'individus pour chaque allèle.
    • Simulation déjà initialisée : Génère une nouvelle génération à partir de couples pris au hasard parmi les membres acifs de la génération actuelle.
  3. Démarrage de la simulation automatique : Démarre la simulation automatique en générant une nouvelle génération toutes les secondes jusqu'à atteindre la génération indiquée dans Nombre de générations.
  4. Paramétrage de la simulation : Configuration dynamique de la simulation :
    1. Nombre d'élément à chaque génération : Configure le nombre d'individus à générer pour la/les prochaine(s) génération(s) : actif en mode simple pas ou automatique,
    2. Nombre de générations : Nombre de générations à simuler en mode automatique.
    3. Ajouter des mutations aléatoires : Si coché, cette case permet d'ajouter des mutation au cours de la nouvelle génération (et donc de voir des enfants exprimer des allèles qui ne dépendent pas de celles présentes chez les parents),
      lorsque cette case est cochée, un curseur permet de régler le taux de mutation, en pourcentage de probabilité d'avoir une mutation lors de la prochaine génération.
  5. Graphique d'évolution de la population : Graphique indiquant l'évolution du nombre d'individus pour chaque allèle au cours des générations (les allèles sont repérées par les mêmes couleurs que celles des blobs),
  6. Numéro de la dernière génération : Numéro de la dernière génération simulée
  7. Individus de la dernière génération : Représentation graphique de la dernière génération simulée à partir de blobs de couleurs.
    Note : Il est possible, en mode pas-à-pas, de simuler la sélection naturelle (mort d'individus avant leur reproduction au cours d'une génération) en cliquant sur les individus qui possèdent un allèle désavantageux dans un milieu donné.

III.2. Démarrer la simulation

Pour effectuer une simulation rapide :

  1. Sélectionnez le nombre d'individus à générer à chaque générations, ainsi que le nombre de générations que vous voulez simuler à partir de la zone Paramétrage de la simulation,
  2. Démarrez la simulation en mode "automatique" avec le bouton Démarrage de la simulation automatique
