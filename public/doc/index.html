I. Installation

Gen-O-Mix is an on-line software based on Node.JS. Part of the code is running on a server (web service, random subjects generator...) and thus has to be installed if you want to run your own server.
To do so, you just have to download the source code from the GIT repository.

Once you have the code and Node.JS installed on your computer, just run the command

node bin/www

You can then connect to your server via your webbrowser using URL http://your_ip_address:3000/
